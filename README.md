**Posting options**

*Online:*
Use a PNG with a good resolution for SoMe posts. A PNG can be generating by exporting from the SVG in e.g. Inkscape.
- [Opslagstavlen DTU](https://www.facebook.com/opslagstavlendtu/followers). Only posts by admins ([PF](https://www.pf.dk/studieliv/sekretariat/reklamer-til-studerende)), contact them for posting. 
- [DTU Students](https://www.facebook.com/groups/studentsdtu). We can post directly here, but be sure to look at group rules first.

*Physical poster:* Use PDF printed in A4 or A3 paper format.  You can export a PDF from Inkscape. See [here](https://www.inside.dtu.dk/-/media/dtu-inside-2025/campus/konferencer-events-og-besoeg/campus_service_skilte_plakatpolitik_a4.pdf) for DTU's poster policy.

Be sure to check the PDF after exporting - you might need to select the option 'convert text to paths' to get a working PDF.
- [PF notice boards](https://www.pf.dk/studieliv/sekretariat/reklamer-til-studerende) are administered by PF. The maximum posting duration is one month, and the removal date must be stated on the poster (cf. guidelines above). PF offers a service where they post and remove posters for a fee.

*Flyers:* The current design hasn't been tested in A5 papaer format, but otherwise we can use A4. Use these at events etc.
